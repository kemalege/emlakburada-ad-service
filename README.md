# Emlakburada Ad Service

## Running the Service

The Gateway Service runs on `http://localhost:8080`.

## API Endpoints

### Ads

#### Place a New Ad
**POST** `/ads`

#### Change Ad Status
**POST** `/ads/change-status`

#### Get All Ads
**GET** `/ads`

#### Delete an Ad
**DELETE** `/ads/{id}`

#### Update an Ad
**POST** `/ads/{id}`

#### Get Ad by Id
**GET** `/ads/{id}`
